# 网络安全学习笔记仓库
## 一、项目概述
本仓库是个人网络安全自学专用资料库，所有学习内容统一使用Markdown格式编写，方便阅读、复制命令、导出PDF/HTML、长期归档保存。现阶段学习重心为Web渗透测试，优先梳理SQL注入相关知识点，核心记录sqlmap工具完整用法、实操命令、漏洞检测流程，后续会持续补充端口扫描、目录爆破、漏洞利用等其他Kali渗透工具笔记。

## 二、仓库管理方式
1. 文件托管：所有`.md`笔记文件存放于代码仓库，支持在线新增、编辑、重命名、移动分类，可创建独立文件夹区分不同工具、不同漏洞模块，结构清晰便于检索；
2. 备份方案：仓库文件同步备份至飞牛NAS网络存储，实现云端仓库+本地NAS双重备份，避免学习资料丢失；
3. 内容规范：全部工具参数采用统一Markdown表格格式整理，参数、功能、实操示例一一对应，复制即可直接在Kali终端使用，适配实验报告书写需求。

## 三、核心学习内容
1. 工具基础介绍：对每一款渗透工具撰写概述，说明工具定位、适用场景、核心能力；
2. 完整参数汇总：分类整理工具全部高频参数，搭配通用命令示例，脱离靶场、通用性强；
3. 实操流程记录：记录漏洞判断思路、工具使用完整步骤、常见报错处理方式；
4. 配套实验笔记：记录靶场复现过程、截图说明、漏洞原理、利用结果。

## 四、使用环境与配套工具
1. 操作环境：Kali Linux（预装sqlmap、nmap、dirsearch等全套渗透工具）；
2. 笔记编写：飞牛NAS内置Markdown在线编辑器，无需图形化桌面即可撰写整理；
3. 文件存储：代码托管仓库 + 飞牛NAS双端存储，兼顾在线编辑与本地离线备份。

## 五、项目学习目标
系统化梳理Web安全渗透测试知识点，搭建一套完整、可复用的个人学习笔记库，规范实验报告书写格式，熟练掌握各类渗透工具语法与实操逻辑，加深对SQL注入等Web漏洞原理与利用方式的理解，为后续内网渗透、漏洞挖掘学习打下基础。
