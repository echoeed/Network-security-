# 网络安全学习笔记仓库
## 一、项目概述
本仓库为系统化网络安全自学资料库，全部内容采用标准Markdown格式撰写，支持命令复制、格式导出、长期归档检索。整体学习以Kali Linux系统为核心，系统整理内置各类渗透工具的理论、参数、实操命令，覆盖信息收集、Web渗透、暴力破解、漏洞框架、流量分析、内网渗透等完整攻防流程，内容不局限于单一工具。后续学习路线将延伸至等级保护、安全运维、漏洞挖掘、安全合规等企业级网络安全方向，持续补充配套理论与实操笔记，可供所有网络安全入门学习者参考查阅。

## 二、仓库文件管理规范
1. 分类存储：通过仓库文件夹划分独立模块，按照工具类型、漏洞类别、安全发展方向建立分层目录，例如信息收集工具、Web渗透工具、内网渗透工具、等保合规文档等，层级清晰，方便检索复习；
2. 内容规范：所有工具参数统一采用「参数|作用|示例」Markdown表格记录，命令使用代码块高亮展示，可直接复制至Kali终端运行，适配实训报告、实验文档的书写需求；
3. 版本管理：依托代码仓库实现文档在线新增、编辑、移动、修改，完整留存每一次笔记更新记录，便于追溯学习过程。

## 三、仓库收录内容范围
### 1. Kali Linux 渗透工具实操笔记
覆盖Kali预装全套安全工具：信息收集类（nmap、dirsearch、whatweb、sublist3r）、Web渗透类（sqlmap、burpsuite、xsser、wpscan）、暴力破解类（hydra、hashcat、medusa）、综合漏洞框架（metasploit-framework）、流量抓包分析（wireshark、tcpdump）、Linux主机提权、端口转发隧道工具等，每份工具文档包含工具简介、完整参数对照表、通用实操命令。

### 2. Web渗透测试实战内容
记录SQL注入、XSS跨站脚本、目录遍历、后台路径爆破、CMS漏洞等主流Web漏洞，包含漏洞原理、漏洞识别判断方法、完整漏洞利用流程、靶场复现实操步骤。

### 3. 中长期拓展学习内容
1. 内网渗透：端口转发、内网横向移动、权限维持、哈希抓取、内网欺骗等内网攻防全套知识点；
2. 等保合规相关：网络安全等级保护2.0标准细则、等保测评完整流程、常见安全风险整改方案、安全设备配置规范；
3. 安全运维：防火墙策略配置、系统日志审计、批量漏洞扫描、服务器主机加固、网络安全应急响应基础流程。

## 四、推荐使用环境
1. 实操环境：Kali Linux，官方预装绝大多数渗透测试工具，是Web与内网攻防实操的标准环境；
2. 笔记编辑：任意支持Markdown语法的编辑器，支持代码块、表格、图片插入即可用于笔记整理。

## 五、项目学习目标
以Kali渗透工具实操作为学习基础，搭建一套完整规范、可复用的公共网络安全知识库，帮助入门学习者熟练掌握Web攻防、主机漏洞利用实操技能；后续逐步深入等级保护、安全合规、企业安全运维相关知识，打通从渗透测试到安全测评、安全运维的完整学习路线，形成攻防实操+合规测评一体化的知识体系。
